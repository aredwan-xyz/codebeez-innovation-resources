## Learning Roadmap by Goal

### Goal: Become an AI Engineer (6 months)

```
Month 1: Python + Math foundations
  - Python: Zero to Mastery or CS50P
  - Math: Linear algebra (3Blue1Brown), calculus basics

Month 2: ML Foundations
  - fast.ai Practical Deep Learning (Part 1)
  - OR: Andrew Ng's ML Specialization

Month 3: Deep Learning
  - Andrej Karpathy: Neural Networks: Zero to Hero (full series)
  - Build: character-level language model from scratch

Month 4: LLM Engineering
  - Full Stack LLM Bootcamp
  - HuggingFace NLP Course
  - Build: RAG app with LlamaIndex

Month 5: Production
  - MLOps basics: FastAPI, Docker, W&B
  - Deploy your app to cloud
  - Build: end-to-end AI product

Month 6: Specialization
  - Pick: Vision / NLP / RL / Safety
  - Contribute to open source
  - Build portfolio project
```

### Goal: Understand AI Without Coding (2 months)

```
Week 1-2: Foundations (No code)
  - Lex Fridman Podcast: 3 recent AI episodes
  - Read: "Co-Intelligence" by Ethan Mollick
  - Try: Claude, ChatGPT, Gemini, Midjourney

Week 3-4: Applications
  - Read: "The Coming Wave" by Suleyman
  - Explore AI tools in your industry
  - Complete: "AI for Everyone" (Coursera, free)

Month 2: Strategy
  - Read: "AI Superpowers" by Kai-Fu Lee
  - Listen: "Lex Fridman Podcast #369" (Sam Altman)
  - Build: No-code AI automation (Zapier/Make)
```
