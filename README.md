### Key Techniques

```markdown
# Core Techniques
├── Zero-shot prompting         — No examples, just ask
├── Few-shot prompting          — Provide 2-5 examples
├── Chain-of-Thought (CoT)      — "Think step by step"
├── Tree of Thoughts            — Explore multiple reasoning paths
├── ReAct                       — Reason + Act (tool use)
├── Retrieval-Augmented Gen     — Ground with external knowledge
├── Self-Consistency            — Multiple samples + majority vote
├── Constitutional AI           — Principle-based self-critique
└── Structured outputs          — JSON/XML format constraints
```
