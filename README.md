```markdown
# Core Techniques

тФЬтФАтФА Few-shot prompting          тАФ Provide 2-5 examples
тФЬтФАтФА Chain-of-Thought (CoT)      тАФ "Think step by step"
тФЬтФАтФА Tree of Thoughts            тАФ Explore multiple reasoning paths

тФЬтФАтФА Self-Consistency            тАФ Multiple samples + majority vote
```
