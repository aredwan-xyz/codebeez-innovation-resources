# Security Policy

This repository is a **curated list of links and educational resources** — it ships no
executable code. The main security concerns are therefore:

- A link that points to a **malicious, compromised, or phishing** site.
- A resource that **misrepresents** what it is in order to mislead readers.
- An issue with the project's **GitHub Actions workflows** or repository configuration.

## Reporting a Vulnerability or Malicious Link

Please **do not** open a public issue for anything you believe is actively harmful
(e.g. a link distributing malware). Instead, report it privately:

- Open a [GitHub Security Advisory](https://github.com/aredwan-xyz/codebeez-innovation-resources/security/advisories/new), **or**
- Email **aredwan.tech@gmail.com** with the details.

For a merely **broken or outdated** link (no security risk), please use the
[Broken Link issue template](https://github.com/aredwan-xyz/codebeez-innovation-resources/issues/new/choose) instead — that helps us triage faster.

### What to include

- The exact location (file and line, or section heading).
- Why you believe it is malicious or unsafe.
- Any supporting evidence (scanner output, screenshots, references).

## Response

We aim to acknowledge reports within **7 days** and to remove or replace confirmed
malicious links promptly. Thank you for helping keep this resource safe for everyone.
